# 圖形化編程入門

這是專為初學者設計的教學，您不需要撰寫任何複雜的程式碼，只要透過拖拉積木，就能輕鬆控制智能小車！

## 第一步：下載與安裝軟體
在開始控制小車之前，我們需要先準備好寫程式的工具。
1. **下載主程式**：請至官方網站下載並安裝最新的圖形化編程主程式。
2. **安裝擴充包**：安裝完主程式後，請下載「MBC-jBot專屬擴充包」，並將其匯入軟體中，這樣才能解鎖控制這台小車的所有積木指令。

> 📥 **軟體下載連結**：
> [點此下載主程式與擴充包](https://drive.google.com/drive/folders/1jJvmSl1R0mK_UXuw2hAYiLdEStOk1EM2?usp=drive_link)
> *(內含主程式安裝檔與專屬擴充包)*

## 第二步：連接小車與電腦
1. 請拿出盒內的 **Type-C 傳輸線**。
2. 將一端插入小車的 Type-C 孔，另一端插入電腦的 USB 插孔（支援 USB 2.0 或 3.0）。
3. 如果您的電腦無法識別小車，請安裝驅動程式：
   * **一般入門者**：請在安裝完圖控軟體後，點擊右上方的「設定」按鈕，並選擇「安裝驅動」。
   * **進階使用者**：如果您的電腦無法識別小車，請下載安裝 [👉 CH340 驅動程式 (支援 Windows / Mac 雙平台)](https://drive.google.com/drive/folders/1IQjFcSHTmrgY4PyfSliELT39hWaP1skf?usp=sharing)。
4. 打開圖形化編程軟體，選擇對應的 COM 序列埠完成連線！

## 第三步：讓小車前進
試著將「前進」積木拖曳到畫面中央的「當 Arduino 啟動」方塊下方。接著，在「前進」積木旁邊，設定數值為 `2` 秒。

![第一支程式](/images/blocks/first_program.webp)

::: tip 🚀 程式執行提醒
程式下載（上傳）完成後，請記得**打開小車電源開關**，並按下車頂的**「啟動按鈕」**，小車才會開始執行您的程式喔！
:::


設定完成後，按下執行，看看小車是不是往前開了呢！
