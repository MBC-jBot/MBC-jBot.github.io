# 3. 感測器與按鈕

感測器是小車的「眼睛」與「觸覺」。透過這些積木，小車可以偵測外界環境並做出反應！

> [!IMPORTANT]
> **配件購買提醒：** VL53L0X 雷射測距模組為選購配件，基礎款套件中並未附贈。若您需要使用雷射測距功能（如智慧防撞），請另外加購此模組！

## 積木總覽

*   **雷射測距積木**：讀取前方 VL53L0X 雷射感測器的數值（單位為毫米 mm）。
*   **紅外線感應器積木**：讀取車頭左側或右側的循線感測器數值。
*   **按鈕積木**：判斷車頂的實體按鈕是否被按下。
*   **電池電壓積木**：讀取目前的電池電壓狀態。

### 💡 教學小提示

*   **持續偵測的秘訣**：請記得，感測器的偵測必須放在「重複無限次」的迴圈裡面！如果沒有迴圈，程式只會在一開機時檢查一次，之後小車就算撞到牆壁也不會停下來喔！
*   **低電壓保護**：雖然您可以自己寫積木檢查電壓，但請放心，MBC-jBot 底層已經內建了保鑣。只要電壓低於 3.2V，底盤的 4 顆燈就會全部閃爍紅燈，提醒您趕快去插 USB 充電！
