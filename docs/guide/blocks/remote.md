# 4. PS2 遙控與伺服馬達

想要讓小車變成一台靈活的遙控車或機械怪獸嗎？這一章會教您如何讀取無線搖桿的訊號，並控制外掛的伺服馬達！

## 積木總覽

*   **PS2 連線狀態積木**：檢查接收器是否插妥、手把是否開機連線。
*   **PS2 按鍵偵測積木**：判斷指定的按鈕（如：叉號、圈圈、上下左右）是否被按住。
*   **PS2 搖桿數值積木**：讀取類比搖桿（香菇頭）的推動幅度（數值 0~255）。
*   **伺服馬達積木**：控制擴充腳位 1 號或 2 號伺服馬達轉動到指定角度（0~180 度）。

## 🏆 挑戰範例：無敵遙控機械臂

在這個範例中，我們要寫出一個保護程式：「**先確認搖桿有連線，如果按下叉號鍵，就把 1 號伺服馬達轉到 90 度！**」

程式下載（上傳）完成後，請記得**打開小車電源開關**，並按下車頂的**「啟動按鈕」**，小車才會開始執行您的程式喔！
:::


### ⚠️ 常見錯誤提醒

*   **忘記檢查連線狀態**：在讀取任何按鈕之前，請務必像範例一樣，先用「如果 PS2 是否已連接」把程式包起來。如果接收器不小心掉下來，而程式又一直強制去讀取按鈕，可能會導致小車的通訊陷入混亂！
*   **4 號燈號救星**：如果您的遙控車不會動，請先看小車屁股的 **4 號燈**。黃燈閃爍代表沒連上（可能是沒插接收器或手把沒電），綠燈常亮才是正常待命，按下按鈕時會變成藍燈喔！
