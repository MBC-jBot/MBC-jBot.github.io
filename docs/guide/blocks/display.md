# 2. 聲光與螢幕顯示

MBC-jBot 不僅能跑，還能透過炫麗的 RGB 燈光與 OLED 螢幕與您互動！

> [!IMPORTANT]
> **配件購買提醒：** SSD1315_OLED 螢幕模組為選購配件，基礎款套件中並未附贈。若您需要使用螢幕顯示功能，請另外加購此模組！

## 積木總覽

*   **LED 燈光積木**：可以設定底盤 1~4 號燈的顏色，或是選擇關閉所有燈光。
*   **OLED 顯示積木**：可以清除螢幕，或在指定的座標 (X: 0~15, Y: 0~3) 上顯示中英文文字或數字。

### ⚠️ 常見錯誤提醒

*   **OLED 螢幕超出範圍**：OLED 螢幕的解析度有限。X 座標（左右）只能填寫 `0` 到 `15`，Y 座標（上下）只能填寫 `0` 到 `3`。如果您輸入了太大的數字，文字會超出螢幕邊界而看不見喔！
*   **OLED 疊字問題**：如果您要在同一個位置顯示新的文字，記得先使用「清空 OLED 螢幕」積木，否則新舊文字會疊加在一起變成亂碼。
