# 1. 馬達與精確移動

這一章我們將學習如何使用積木來控制 MBC-jBot 的兩顆強力馬達，讓小車能隨心所欲地在地板上移動！

## 積木總覽

這類積木主要負責控制車輪的速度與停止狀態。

*   **雙輪控制積木**：可以同時給定左輪與右輪的速度（範圍 -100 到 100）。
*   **單輪控制積木**：單獨控制左輪或右輪。
*   **馬達煞車積木**：可以選擇「滑行」或「煞停 (BRAKE)」。

### 💡 教學小提示

*   **為什麼要加等待？**：程式的執行速度非常快（千分之一秒）。如果您放了「前進」接著馬上放「煞車」，小車根本來不及動就會停下來。所以我們必須用「等待」積木來讓小車前進一段時間。
*   **滑行 vs 煞停**：當您選擇「滑行」時，馬達會斷電，但車子會因為慣性再往前溜一點點。如果選擇「煞停」，馬達會利用電磁短路瞬間鎖死輪胎，這在避障比賽中非常重要！
