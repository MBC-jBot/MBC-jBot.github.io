# 6. 下載第一個程式試試看吧！

恭喜您完成了前面的基礎教學！現在，我們準備了一個完整的「**PS2 遙控車與全方位控制**」範例程式。

這個程式結合了我們前面學過的所有技巧：
*   🏎️ **馬達控制**：前後左右全方位移動
*   🎮 **PS2 手把**：類比搖桿與按鍵雙重控制
*   🖥️ **序列埠監控**：即時回傳馬達數值

## 📥 範例程式下載

您可以點擊下方連結，下載我們為您準備好的完整範例程式：

> 🎁 **[點此下載 PS2 遙控範例程式 (PS2_demo.ob)](/example/PS2_demo.ob)**

### 如何使用這個檔案？
1. 下載上方的 `PS2_demo.ob` 檔案到您的電腦中。
2. 打開 MBC-jBot 圖形化編程軟體。
3. 點擊左上角的 **「檔案」** -> **「從電腦中挑選」**。
4. 選擇您剛剛下載的 `PS2_demo.ob`。
5. 連接小車，點擊上傳即可馬上開始遙控！
