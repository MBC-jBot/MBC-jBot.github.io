# 環境安裝與初始化

本章節將引導您如何在 Arduino IDE 中設定 MBC-jBot，並寫下您的第一支 C++ 程式。

## 1. 環境準備
MBC-jBot 的核心是一顆 `ATmega328P`，這意味著它與經典的 **Arduino Uno** 完全相容！（小車晶片已內建 Arduino Uno 的 bootloader）
*   **開發板設定 (Board)**：請在 Arduino IDE 中選擇 `Arduino Uno`。
*   **核心庫 (Core)**：使用官方的 `Arduino AVR Boards` 即可。

> 📥 **JBC_Bot 程式庫下載**： 
> [點此下載 C++ 程式庫](https://drive.google.com/drive/folders/1jJvmSl1R0mK_UXuw2hAYiLdEStOk1EM2?usp=drive_link) 
> (下載後請解壓到程式碼資料夾內)

## 2. 引入 JBC_Bot 程式庫
要在程式中控制小車，您必須在檔案的最上方引入 `JBC_Bot` 程式庫：

```cpp
#include <JBC_Bot.h>
```
這行程式碼會自動建立一個名為 `Bot` 的全域物件，您後續所有的操作都會透過這個 `Bot` 來呼叫。

## 3. 系統初始化 (`begin()`)

在 `setup()` 區塊中，您**必須**呼叫 `Bot.begin()` 來啟動小車的所有硬體。

> [!WARNING]
> **安全啟動機制 (重要！)**
> `Bot.begin()` 內部包含了一個**強制阻擋**的機制。當程式執行到這裡時，系統會**暫停所有程式碼的執行**，直到使用者**按下車體上方的實體按鈕**。
> 
> 💡 **貼心提示：** 在按鈕被按下之前，主機按鈕上方的 **3號燈號會閃爍黃燈**，藉此提醒使用者「我已經準備好了，按我出發！」。這個機制的目的是防止小車在一開機、尚未放到安全地面時就意外暴衝！

### 基本程式碼架構範例

```cpp
#include <JBC_Bot.h>

void setup() {
  // 啟動所有感測器與馬達，並等待使用者按下實體按鈕
  Bot.begin();
  
  // 這裡可以放您的其他初始化程式碼
  Serial.begin(9600);
}

void loop() {
  // 您的主要控制邏輯
}
```

## 4. 低電壓保護與自動充電
MBC-jBot 內建了電池電壓偵測（透過內部 ADC 腳位）。
當您呼叫任何控制馬達或感測器的函式時，系統會在底層自動檢查電壓。如果發現電池電壓過低（低於約 3.2V），小車底盤上的 **4 顆 RGB 燈號會全部一起閃爍紅燈**，強烈提醒您該充電了！

### 如何充電？
非常簡單！這台小車支援 **USB 即插即充**。
1. 將 USB 線插上小車，另一頭接上電腦或手機充電器。
2. 觀察小車**右下方的 USB 孔左側**：
   *   🔴 **紅燈亮起**：代表正在充電中。
   *   🟢 **綠燈亮起**：代表電池已經完全充飽！

如果想手動在程式裡讀取當前電壓，可以使用：
```cpp
uint16_t voltage = Bot.getBatteryVoltage(); // 回傳轉換後的電壓數值
```

> [!WARNING]
> **電壓量測注意事項**
> 如果您在使用 USB 線連接電腦時呼叫這個函式，讀取到的數值會是 **USB 的供電電壓**（通常在 5V 左右），而不是真實的電池電壓！
> 
> 若您想準確查看電池的剩餘電壓數值，建議您**不要連接 USB 電源**，並將這個電壓數值透過 **OLED 螢幕** 顯示出來觀察。
