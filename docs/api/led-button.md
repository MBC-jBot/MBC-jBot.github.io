# 聲光與互動

MBC-jBot 除了能跑能偵測，還具備了豐富的視覺與觸覺回饋機制。

## RGB 炫彩燈條 (WS2812B)

小車底盤搭載了多顆可程式化的 WS2812B 炫彩 LED 燈。這些燈號除了會在低電壓時自動閃爍警告外，您也可以手動控制它們發出任何您想要的顏色！

### 1. 設定指定位置的燈光顏色 (`setLED`)

參數說明：
*   `index`: 燈號的編號 (目前開放給使用者自訂的編號為 0, 1, 2, 3)。
    * `0`：左前方循線感測器旁的燈
    * `1`：右前方循線感測器旁的燈
    * `2`：主機按鈕上方的燈 (3號燈)
    * `3`：主機後方的燈 (4號燈，通常用於 PS2 狀態指示)
*   `r`, `g`, `b`: 紅、綠、藍三原色的亮度 (範圍從 `0` 到 `255`)。

```cpp
// 語法：Bot.setLED(編號, 紅色值, 綠色值, 藍色值);

// 將 0 號燈設為純紅色 (最亮)
Bot.setLED(0, 255, 0, 0);

// 將 2 號燈 (按鈕燈) 設為紫色 (紅+藍)
Bot.setLED(2, 150, 0, 150);
```

> [!TIP]
> **極速的燈光控制**
> MBC-jBot 底層對燈光訊號進行了極致的最佳化，確保傳送燈號顏色時非常快速，完全不會影響到主程式或是馬達的運作！

### 2. 關閉所有手動燈光 (`clearLEDs`)

如果您想一口氣關閉所有自訂的燈光，並把燈條控制權交還給系統（例如交還給循線感測器或按鈕的自動連動功能），可以呼叫這個函式。

```cpp
Bot.clearLEDs();
```

## 實體按鈕偵測

車體上方有一顆實體按鈕。除了在開機時用於解除安全鎖定外，在主程式運作期間，您也可以讀取這顆按鈕的狀態來觸發特定動作。

### 讀取按鈕狀態 (`isButtonPressed`)

回傳一個布林值 (`bool`)。如果按鈕被按著不放，就會回傳 `true`，否則回傳 `false`。

```cpp
// 語法：bool Bot.isButtonPressed();

if (Bot.isButtonPressed()) {
  // 當按鈕被按下時，讓小車緊急煞車
  Bot.setMotors(BRAKE, BRAKE);
  
  // 也可以亮起警示燈 (使用左前方的 0 號燈為例)
  Bot.setLED(0, 255, 0, 0); 
}
```

> [!NOTE]
> **按鈕狀態指示燈與控制權優先順序**
> 當按鈕被按下時，系統會自動接管第 3 顆 LED 燈 (編號 `2`)，讓它發出黃光作為視覺回饋。
> 
> 若您在程式中已經透過 `Bot.setLED(2, ...)` 指派了特定顏色給這顆燈，當按鈕壓下時，它並不會被強制蓋掉，而是會像警車閃燈一樣，**在您的自訂顏色與黃光之間交替閃爍**！當按鈕放開後，它就會自動恢復成純顯示您的自訂顏色。這跟 PS2 遙控器與循線感測器接管燈號的機制完全相同！
